# counterEVM
**Counterfactual explanations in Earned Value Management**


This repository contains the scripts that reproduce the results presented in our paper:
ACTIONABLE DECISION-MAKING IN PROJECT MANAGEMENT CONTROL USING COUNTERFACTUAL EXPLANATIONS 

## Files in this repository:

* Monte_Carlo_Simulation.ipynb Monte Carlo simulation of the Fuel Tank Filter project PERT/CPM network
* ModelSelection.ipynb contains classification model selection and model performance assessment
* Selection_of_instance.ipynb Selection of a specific instance for which alternative scenarios will be constructed
* CFs_generation.ipynb Generation and Interpretation of Counterfactuals 
* Plots.ipynb CFs comparison plots
